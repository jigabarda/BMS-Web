# Web Frontend Broadcast Management System
💻 Full-Stack Broadcast Web Dashboard

✨ Description

The Web Frontend is the admin dashboard for the Full-Stack Broadcast System. Built with React and Next.js, it allows administrators to create, manage, and send broadcasts seamlessly. Broadcasts appear in real time on mobile devices, ensuring timely updates. The dashboard is fully responsive with Tailwind CSS, and uses JWT authentication to communicate securely with the backend. Components are reusable, and state management is handled via React Hooks for efficient API calls.

# 🛠 Tech Stack & Tools Used

⚛️ React – Frontend library

🌐 Next.js – Server-side rendering framework

🎨 Tailwind CSS – Responsive styling

🏗 React Hooks – State and lifecycle management

🛣 Next.js App Router – Navigation and routing

🌐 fetch / Axios – API integration

🔑 JWT Authentication – Secure API requests

🔥 Firebase

🔔 Real-time push notifications to mobile clients

⚡ Instant updates for new broadcasts

⚛️ React

🧩 Modular and reusable components

⚡ Smooth UI transitions and responsive design

# ⚙️ Installation & Run

Clone the repository:

git clone <repo-url>
cd bms-frontend


Install dependencies:

npm install


Start the development server:

npm run dev


Open in the browser: http://localhost:3000
